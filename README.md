# Benkei for Windows

Benkei for Windowsは、Windowsで薙刀式カナ入力を実現するための常駐ソフトです。
薙刀式は大岡俊彦氏が開発されたカナ入配列です。
アイコン画像の権利は大岡俊彦氏に帰属します。

- Windows nativeで、薙刀式専用とすることで、薙刀式の設計意図を実現し、高速かつ安定して動作する。
- Windowsの設定変更や、 IMEのキー割り当て変更をしなくても動作する。


### 薙刀式の仕様

- [x] ctrl+shift+0で薙刀式をオフにして通常の入力へ戻る（サスペンド）、
- [x] ctrl+shift+1で再開。

- [x] IMEオフ（半角、英数直接入力モード）は、FG同時押し。
- [x] IMEオン（全角、カナ漢字変換モード）は、HJ同時押し。

- [x] ctrl, win, altを押している時はqwerty配列に戻る。
- [x] SandS（シフトキーとスペースキーを兼ねる）を使用。
- [x] MV同時押しにエンター。

- [x] 単打面、シフト面
- [x] その他同時押し面
- [x] 編集モード
- [x] 固有名詞モード
